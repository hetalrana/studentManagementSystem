## About Student Management

<p> Here I have managed student registration and login system where user can add/edit/delete and view subjects.</p>

## System requirements

<p>xampp or wamp with PHP 8.2</p>
<p>composer setup for package dependencies</p>

## Steps to setup project

<p>1. download zip from this repository. </p>
<p>2. Need to run composer command 'composer install' </p>
<p>3. Import sql file from project's public folder for data management in student_management_db</p>
<p>4. Run "php artisan serve" command. </p>
<p>5. open login link http://127.0.0.1:8000/login or registration link http://127.0.0.1:8000/register</p>
